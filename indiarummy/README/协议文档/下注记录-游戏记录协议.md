#### 下注记录-游戏记录协议

1、下注记录
```
req: {
    c:193,
    uid:10101,
    gameid:19
}
```
res:
```
{
    c:193,
    code:200,
    gameid:19,
    records:[
        {
            i: 'AB221019346571230002', --序列号
            t: 1692345274, --时间戳
            bet: 500, --押注
            win: 100, --赢钱
            result: {...}, --开奖结果json（牌类无）
            p: 1, --押注区域（牌类无）
        },
        ...
    ]
}
```
