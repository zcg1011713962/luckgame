#### 邀请协议

##### 1、邀请基本信息

Req:

```
{
	"c":254,
	"uid":100100,
}
```

Resp:

```
{
	"c": 254,
    "code": 200，
    "info": {
		"invite": {
            "coin1": 50, --1级下级绑码 可获得的金币奖励
            "coin2": 20 --2级下级绑码 可获得的金币奖励
        },
        "bet": {
            "rate1": 0.05, --1级下级下注，可以获得的佣金比例
            "rate2": 0.02 --2级下级下注，可以获得的佣金比例
        },
        "recharge": {
            "rate1": 0.05, --1级下级充值，可以获得的佣金比例
            "rate2": 0.02 --2级下级充值，可以获得的佣金比例
        },
        "code": "62506", --我的邀请码
        "status": 1, --我是否还可以绑定上级的邀请码, 1:是 0：否
        "url": "www.baidu.com/?code=62506", --我的邀请链接
        "total_bonus": 0,  -- total bonus
		"reg_bonus": 0, -- 下级注册获得的bonus
		"total_refers": 0, --邀请人数
        
    }
}

```

##### 2、绑定邀请码

用户绑定上级邀请码

错误码:

```
['INVITE_HAD_BINDED'] = 1452, --已经绑定过邀请码
['INVITE_BIND_SELF'] = 1453, --不能绑定自己的邀请码
['INVITE_BIND_IN_CIRCLE'] = 1454, --不能绑定下级的邀请码
['INVITE_BIND_ERR'] = 1455, --绑定错误
```

Req

```
{
	'c':257,
	'uid':100100, --用户uid
	'code':'123456', --上级的邀请码
}
```

Resp:

```
{
	'c':257,
	'uid':100100,
	'code':200,
	'spcode':0
}
```

##### 3、我的下级列表

Req:

```
{
	'c':255
	'uid':100100
}
```

Resp:

```
{
	'c':255,
	'code':200,
	'spcode':0,
	'data':[
			{
          'uid' : '100101', --下级uid
          'playername' : 'xxxxx', --下级用户名,
          'usericon' :  '下级头像地址',
          'svip' : 'svip',  --下级的vip等级
          'svipexp' : 2320, --下级的vip经验值
          'times': 123, --下级的下注次数
          'bet' : 300200, --下级的下注金币
		  'recharge: 0, --下级的重置金币数
          'login_time': 12313123123, --下级最后登录时间戳
          'coin' : 1000, --我获得的佣金
      },
	...
	]
}
```

##### 5、我的收益列表

Req:

```
{
	'c':256,
	'uid':'100100',
}
```

Resp:

```
{
	'c':256,
	'code':200,
	'spcode':0,
	'data': [
			{
				'create_time':13432423, --日期时间戳
				'bet': 200, --总下注
				'recharge':0, --当日下级充值
				'coin': 100, --日期对应的总收益
			},
			...
	]
}
```

##### 6、我某天的收益详情

Req:

```
{
	"c":259,
	"uid":199199,
	"time":186344343,
}
```

Resp:

```
{
	"c":259,
	"code":200,
	"spcode":0,
	"data":[
		{
		    uid = 100200, --下级UID
			playername = "下级昵称",
			usericon = "下级头像地址",
			svip = "下级vip等级",
			svipexp = "下级VIP经验值",
			create_time = row.create_time, --日期时间戳,需要客户端转为日期
			bet = 2000, --下级在这1天的总下注
			recharge = 10, --下级在这1天的总充值
			coin = 10000, --该下级该日对应的总佣金
		},
		...
	],
}
```
